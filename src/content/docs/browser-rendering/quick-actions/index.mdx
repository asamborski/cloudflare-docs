Quick Actions provide simple HTTP endpoints for common browser tasks like capturing screenshots, extracting HTML content, generating PDFs, and more.

The following are the available options:

<DirectoryListing />

Use Quick Actions when you need a fast, simple way to perform common browser tasks such as capturing screenshots, extracting HTML, or generating PDFs without writing complex scripts. For more advanced automation, custom workflows, or persistent browser sessions, use [Puppeteer](/browser-rendering/puppeteer/), [Playwright](/browser-rendering/playwright/), or [CDP](/browser-rendering/cdp/).

## Before you begin

Before you begin, make sure you [create a custom API Token](/fundamentals/api/get-started/create-token/) with the following permissions:

- `Browser Rendering - Edit`

:::note[Note]

You can monitor Browser Rendering usage in two ways:

- In the Cloudflare dashboard, go to the **Browser Rendering** page to view aggregate metrics, including total Quick Actions requests and total browser hours used.
  <DashButton url="/?to=/:account/workers/browser-rendering" />
- `X-Browser-Ms-Used` header: Returned in every Quick Actions response, reporting browser time used for that request (in milliseconds).

:::

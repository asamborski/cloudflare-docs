Dashboard log streams are only available for <GlossaryTooltip term="remotely-managed tunnel">remotely-managed tunnels</GlossaryTooltip>. To view logs from the dashboard:

1. In [Cloudflare One](https://one.dash.cloudflare.com/), go to **Networks** > **Connectors** > **Cloudflare Tunnels** and select a tunnel.
2. In the sidebar, select the **Connector ID** for the `cloudflared` instance you want to view.
3. Select **Begin log stream**.
